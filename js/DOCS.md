# Aleo SDK

#### A Software Development Kit (SDK) for Zero-Knowledge Transactions.

This kit provides the following structures to work with:

1. [Private Key](/PrivateKey)
2. [View Key](/ViewKey)
3. [Record](/Record)
4. [Address](/Address)
5. [Signature](/Signature)

Happy hacking!
